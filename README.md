# 專案架構與流程

## 架構圖

+---------------------+        +---------------------+       +---------------------+
|                     |        |                     |       |                     |
|      前端 (React)    | <----> |      後端 (Node.js)  | <----> |    資料庫 (MongoDB)  |
|                     |        |    (Express.js)     |       |                     |
+---------------------+        +---------------------+       +---------------------+
          |                            |                             |
          |                            |                             |
          v                            v                             v
+---------------------+        +---------------------+       +---------------------+
|                     |        |                     |       |                     |
|  用戶界面 (表單, 顯示學生名單)  |        |    路由設定, CRUD 功能  |       | 存儲學生資料，查詢、更新、刪除 |
|                     |        |    API (增、刪、改、查)  |       |                     |
+---------------------+        +---------------------+       +---------------------+



## 流程圖

1. 用戶從前端表單輸入學生資料
2. 前端將資料送到後端 API
3. 後端接收資料並進行新增操作
4. 資料庫將學生資料保存到 MongoDB
5. 後端返回成功訊息
6. 前端更新學生名單顯示
